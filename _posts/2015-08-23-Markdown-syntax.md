## 3. 快速在jekyll中插入图片：

	! \[num](/path/to/image "title")

如： ![有个姐姐的好处](/images/201508/sister.gif "有个姐姐的好处")

## 4. 代码插入
在一般的段落文字中，你可以使用反引号 ` 来标记代码区段；如果要建立一个已经格式化好的代码区块，只要每行都缩进 4 个空格或是一个 tab 就可以了。（这里用两个tab）

	Code here.

或者用三个反引号包围。
